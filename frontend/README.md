## Project Setup

Create a `.env` file in the project root and add:

```
VUE_APP_STRIPE_PUBLIC_KEY=
```

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
